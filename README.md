# Fake News Detection System

## Overview

This project is a Machine Learning-based Fake News Detection System that classifies news articles as **Real** or **Fake**. It uses Natural Language Processing (NLP) techniques along with the TF-IDF feature extraction method and a Passive Aggressive Classifier to make predictions.

## Features

- Detects whether a news article is Real or Fake
- Text preprocessing using NLP
- TF-IDF feature extraction
- Machine Learning-based prediction
- Google News verification
- Simple Gradio web interface

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- NLTK
- Gradio
- Sentence Transformers

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/MdZafar-07/Fake-News-Detection-System.git
```

2. Open the project folder.

```bash
cd Fake-News-Detection-System
```

3. Install the required libraries.

```bash
pip install pandas numpy scikit-learn nltk gradio sentence-transformers feedparser newsapi-python
```

4. Open `Fake_News_Detector_Final.ipynb` in Jupyter Notebook or Google Colab.

5. Run all the cells.
